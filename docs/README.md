## 🎯 文档使用指南

### 新手入门（30分钟）

1. **阅读快速参考**（5分钟）
   - 了解项目结构
   - 掌握常用命令

2. **查看 API 文档**（10分钟）
   - 理解请求格式
   - 测试 API 调用

3. **尝试部署**（15分钟）
   - 本地开发或 Docker 部署
   - 验证功能

### 深入开发（2-3小时）

1. **阅读架构设计**
   - 理解系统设计
   - 掌握核心模块

2. **查看重构记录**
   - 了解演进过程
   - 学习设计模式

3. **阅读测试指南**
   - 理解测试策略
   - 学习测试编写

### 生产部署（1小时）

1. **阅读部署指南**
   - 选择部署方案
   - 按步骤操作

2. **参考快速参考**
   - 检查配置
   - 验证部署
